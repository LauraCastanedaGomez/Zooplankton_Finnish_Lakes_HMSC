# Zooplankton_Finnish_Lakes_HMSC
Code and scripts for modelling zooplankton in Finnish lakes using HMSC.

This repository contains data, models, results and scripts for the study of zooplankton communities in Finnish lakes using HMSC (Hierarchical Modelling of Species Communities).

Repository structure
/data        → Raw and processed input data for the analyses
/models      → Unfitted and Fitted HMSC models
/results     → Output of HMSC model fit, cross-validation results and plots
/scripts     → R scripts used for HMSC modelling, variance partitioning and functional coocurrence
