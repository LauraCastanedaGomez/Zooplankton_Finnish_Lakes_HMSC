# Read Me

This repository contains data, models, results and scripts for the study of zooplankton communities in Finnish lakes using HMSC (Hierarchical Modelling of Species Communities).


## Repository structure

/data        → Raw data (.xlsx files) and processed input data (.RData file) for the analyses

/models      → Unfitted and Fitted HMSC models

/results     → Outputs of HMSC model: model fit, cross-validation results, MCMC convergence, parameter estimates and plots

/scripts     → R scripts used for HMSC modelling (S1, S2, S3, S4, S5 and S6), variance partitioning and functional co-occurrence analyses
